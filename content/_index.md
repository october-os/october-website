October Linux is an Arch-based Linux distribution that provides an installer written in Go, which enables a fast and automated setup using a JSON configuration. The installer deploys a complete system including a pre-configured desktop environment and essential applications to deliver an out-of-the-box experience, while remaining as faithful as possible to the minimalist philosophy of Arch.
